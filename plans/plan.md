# Plan: SaveMoney - Full Clone Money Lover
Created: 2026-03-27
Status: 🟡 In Progress

## Overview
App quản lý chi tiêu cá nhân cho Android & iOS.
Flutter + Firebase Auth + Cloud Firestore + Firebase Storage.

## Tech Stack
- Framework: Flutter (Dart)
- Auth: Firebase Auth + Google Sign-In
- Database: Cloud Firestore (offline persistence bật sẵn)
- Storage: Firebase Storage (ảnh chứng từ)
- State: Riverpod
- Charts: fl_chart
- Navigation: Go Router
