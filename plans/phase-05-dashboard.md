# Phase 05: Dashboard & Reports
Status: ⬜ Pending
Dependencies: Phase 04

## Objective
Màn hình tổng quan (dashboard) và báo cáo biểu đồ đẹp.

## Implementation Steps
### Dashboard (Home Tab)
1. [ ] Tổng số dư tất cả ví
2. [ ] Thu / Chi trong tháng (so sánh)
3. [ ] Danh sách giao dịch gần đây (5-10 items)
4. [ ] Pie chart mini: phân bổ chi tiêu theo category

### Reports (Tab Báo cáo)
5. [ ] Chọn kỳ: ngày / tuần / tháng / năm
6. [ ] Bar chart: thu vs chi theo thời gian
7. [ ] Pie chart: chi tiêu theo category
8. [ ] Bảng top category chi tiêu nhiều nhất
9. [ ] So sánh với kỳ trước (tăng/giảm %)

## Packages sử dụng
- `fl_chart` - Bar chart, Pie chart, Line chart

## Test Criteria
- [ ] Dashboard load nhanh (< 1 giây)
- [ ] Biểu đồ hiển thị đúng số liệu
- [ ] Chuyển kỳ báo cáo mượt
